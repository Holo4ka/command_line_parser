**Программа для анализа брендов**

Пример запуска:
`python main.py --files products1.csv products2.csv --report average-rating`

Вывод для примера выше:
+----+---------+----------+
|    | brand   |   rating |
+====+=========+==========+
|  1 | apple   |     4.55 |
|  2 | samsung |     4.53 |
|  3 | xiaomi  |     4.37 |
+----+---------+----------+

Параметры:
- "--files" - список файлов для анализа
- "--report" - тип отчета (поддерживаемые типы: average-rating, average-price)

**Добавление нового отчета**

Для добавления поддержки нового типа отчетов необходимо:
1. Написать функцию, которая подсчитывает нужные метрики
2. При необходимости, написать функцию, которая будет отвечать за корректный вывод таблицы с отчетом
3. Добавить в константу REPORTS данные о новом отчете:
- Ключ словаря REPORTS - название отчета
- Значение этого ключа - словарь с параметрами 'calculator' (функция для подсчета метрик), 'value_name' (название столбца/столбцов), 'printer' (функция для вывода таблицы)
