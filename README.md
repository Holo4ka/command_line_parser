**Программа для анализа брендов**

Пример запуска:
`python main.py --files products1.csv products2.csv --report average-rating`

Вывод для примера выше:

![example](https://github.com/user-attachments/assets/b853dcf9-38e9-429f-bd92-9002290ddce7)

Параметры:
- "--files" - файлы для анализа
- "--report" - тип отчета

Вывод Help-message:
`python main.py -h`

**Добавление нового отчета**
1. Создать функцию для подсчета необходимой метрики
2. При необходимости написать функцию для вывода таблицы (например, если нужно изменить количество столбцов)
3. Обновить словарь REPORTS:
- Ключ словаря REPORTS - название нового отчета
- Значение по этому ключу - словарь с параметрами: __'calculator'__ (функция, которая считает метрики), __'value_name'__ (столбец/столбцы, которые будут присутствовать в выводе), __'printer'__ (функция, которая отвечает за вывод)

Изображения с примерами работы программы находятся в папке examples
